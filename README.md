# analisis_datos_ia
Proyecto académico orientado a la comprensión de los conceptos básicos de la Inteligencia Artificial, su aplicación en la vida cotidiana y el análisis de datos mediante ejemplos prácticos. Se trabajó en el uso de modelos simples para el reconocimiento de patrones y automatización de tareas básicas.
#TECNOLOGAS UTILIZADAS
Python
Google Colab
Herramientas básicas de Inteligencia Artificial
Análisis de datos (nivel introductorio)
#Rol y aportes personales
Participé en el desarrollo de ejercicios prácticos, creación de variables, análisis de resultados y comprensión del funcionamiento de modelos básicos de IA. Aporté en la resolución de problemas y en la interpretación de datos obtenidos.
